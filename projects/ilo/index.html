💻 ilo

This is a simple tool that allows you to input Toki Pona, and it will provide you the definition of each word. It does not understand sentence structure and doesn't give you an actual translation, but provides you information so you can translate yourself.

It is set up as a PWA, so you can also install it to your android device and even use it offline.

  • Source code: https://github.com/0xdstn/ilo/
  • Live version: https://0xdstn.site/ilo/

There is also a firefox addon to add an item to context the menu when you select some text and right click on it. When you click the item it opens a new tab with the selected text in ilo to be translated.

  • Addon: https://addons.mozilla.org/en-US/firefox/addon/ilo/
  • Source code: https://github.com/0xdstn/ilo-firefox
