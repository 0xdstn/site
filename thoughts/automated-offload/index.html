💭 Automated Offload

2023.11.14

I saw Zach's post on overengineering the #100DaysToOffload counter in Jekyll, which made me think "Why am I not automating this?".

My solution isn't as automated as it could be, but now instead of including the actual text for the "x/100 of #100DaysToOffload" in my post as a line, I just add a tag o:001 and it generates it for me.. This is where it's not as automated as it could be, since I have to provide the number.

Using this tag, I also automated the list on my 100 Days to Offload post. I just add the tag f:100daystooffload and it creates an ordered list of all the post.

Honestly it's been really annoying to have to add new posts to the list and copy the x/100 line each time, so this will be nice.
