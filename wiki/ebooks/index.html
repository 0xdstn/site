📘 Ebooks

Below are my notes on managing my ebooks library.

  1. Notes
  2. Ebook management

Notes

I currently own a 10th Generation Kindle.

It is still set up as an Amazon device, but if it becomes possible I may jailbreak it. As of writing this, these instructions only support firmware version <= 5.14.2 and mine is running 5.16.2.1.

Ebook management

Previously I would take ebooks I got outside of Amazon and import them to Amazon via their Send to Kindle feature. I then decided to take more control over my ebook files and removed them from there.

I now take ebooks, convert them to mobi using Calibre, and then add those mobi files to a folder on my computer and in a folder on the Kindle via USB.

Remove DRM

This is the process I took to remove DRM from my books bought on Amazon and Google.

Moving forward I will try to always purchase DRM free books. I found this resource which I will use to find sources for the books I want to buy.

Install Calibre with DeDRM

  1. Download and install Calibre
  2. Download the latest DeDRM_tools and extract DeDRM_plugin.zip from it.
  3. In Calibre go to Preferences > Advanced > Plugins > Load plugin from file and select the DeDRM_plugin.zip file

Kindle

Note: Books published after January 3rd, 2023 will not download, and therefore cannot be converted. Fortunately, I only had one book that fit this as of migrating.

  1. Download and install Kindle for Windows v1.17
  2. In Kindle for Windows, log in to your Amazon account and download all the books you want to convert
  3. In Calibre click Add Books and navigate to the Amazon books (mine were in C:\Users\USERNAME\Documents\My Kindle Content)
  4. In Calibre click select all the books and click Convert books > bulk convert. Select the format you want and click convert. With the plugin installed DRM should already be removed.

Google Books

  1. Go to Google Books
  2. Click on each book you want to download
  3. For each click on the settings cog and click Download EPUB (they will be acsm files)
  4. Follow the Remove DRM from ascm files below

Books-a-Million

  1. Go to My Download Library on Books-a-Million
  2. Click on Download for each book you want to download (they will be acsm files)
  3. Follow the Remove DRM from ascm files below

Remove DRM from acsm files

  1. Download and install Adobe Digital Editions
  2. Open Adobe Digital Editions and log in with your Adobe account
  3. In ADE click File > Add to Library and select the downloaded books. It will have you log in to an Adobe account. This converts the files to epub.
  4. In Calibre click Add Books and navigate to the ADE books (mine were in C:\Users\USERNAME\Documents\My Digital Editions)
  5. In Calibre click select all the books and click Convert books > bulk convert. Select the format you want and click convert. With the plugin installed DRM should already be removed.
